Package org.jscep.client.verification Description

Provides an interface and several implementation classes for verifying the identity of a certificate.

Classes in this package should be combined with CertificateVerificationCallback for verifying the certificate sent from the SCEP server in GetCaCert requests. The SCEP specification recommends that one GetCaCert request is made before each operation, so this package will be used quite frequently.
